# 🎤 AI 기반 맞춤형 포트폴리오 추천 서비스 발표 대본

## 📌 발표 시간: 약 5-7분

---

## 1️⃣ 인사 및 서비스 소개 (30초)

안녕하세요. 오늘 제가 소개드릴 서비스는 **"AI 기반 맞춤형 포트폴리오 추천 서비스"**입니다.

이 서비스는 고객의 투자 성향을 분석하여 개인별 맞춤형 포트폴리오를 제공하고,
은행의 금융 상품까지 연계하여 추천하는 **원스톱 투자 솔루션**입니다.

---

## 2️⃣ 배경 및 필요성 (45초)

최근 MZ세대를 중심으로 주식 투자가 대중화되었지만,
많은 투자자들이 **"무엇을 얼마나 사야 할지"** 고민하고 있습니다.

특히 투자 초보자들은:
- ✅ 종목 선택의 어려움
- ✅ 적절한 분산 투자 방법 부재
- ✅ 리밸런싱 시기 파악 곤란
- ✅ 절세 상품 활용법 미숙

이러한 문제를 해결하기 위해 개발했습니다.

---

## 3️⃣ 핵심 기능 시연 (2분)

### 📊 시작 화면
*[Home 화면 보여주기]*

"포트폴리오 추천 시작하기" 버튼을 클릭하면 설문이 시작됩니다.

### 📝 맞춤 설문조사 (5문항)
*[Survey 화면 진행]*

간단한 5개 질문으로 고객의:
- 투자 금액
- 투자 기간
- 리스크 성향
- 선호 섹터
- 투자 경험

을 파악합니다. 모바일에서도 편리하게 입력 가능합니다.

### 🤖 AI 포트폴리오 생성
*[Results 화면으로 이동]*

**Modern Portfolio Theory**를 기반으로 최적 포트폴리오를 생성합니다.
- 실시간 주가 연동 (pykrx API)
- 섹터별 분산 투자
- 리스크 레벨 맞춤 구성

### 💰 리밸런싱 분석 (핵심 차별점!)
*[리밸런싱 분석 버튼 클릭]*

**3가지 탭으로 구성된 통합 분석:**

1. **주식 리밸런싱**
   - 목표 비중 vs 현재 비중 비교
   - 매수/매도 수량 자동 계산

2. **당행 펀드 추천** ⭐
   - AI 포트폴리오 분석 기반
   - 부족한 섹터 보완 상품 추천
   - 예: 성장주 위주 → 채권형 펀드 추천

3. **ISA 계좌 ETF 추천** ⭐
   - 절세 혜택 극대화
   - 배당 수익률 기반 우선순위
   - 연 400만원 비과세 활용

## 4️⃣ 기술적 특징 (1분)

### 🎯 AI 기술 활용
- **MPT 알고리즘**: 효율적 프론티어 계산
- **포트폴리오 분석 엔진**: 섹터별 가중치 분석
- **보완 상품 매칭**: 포트폴리오 약점 보완

### 📱 기술 스택
- **Frontend**: React 18 + Tailwind CSS
- **Backend**: Flask + Python
- **Data**: pykrx (한국거래소 실시간)
- **배포**: Vercel (글로벌 CDN)

### 🚀 성능 최적화
- 캐싱으로 응답속도 90% 개선
- 모바일 최적화 (터치 타겟 44px)
- 실시간 주가 업데이트

## 💡 예상 질문 및 답변

### Q1. 투자 손실 책임은?
A: 투자 참고 자료이며, 최종 결정은 고객 본인. 리스크 고지 강화.

### Q2. 기존 로보어드바이저와 차이점?
A: 당행 상품(펀드, ISA) 연계가 핵심. 통합 자산관리 플랫폼.

### Q3. 수익 모델은?
A: 펀드 판매 수수료 + ISA 계좌 수수료 + 프리미엄 서비스(향후)

### Q4. 규제 이슈는?
A: 투자 권유가 아닌 정보 제공. 금융투자협회 가이드라인 준수.
