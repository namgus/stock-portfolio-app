## 📋 데모 시나리오

### ⏱️ STEP 1: 고객 상황 설정 (20초)

**발표자 멘트**:
```
"40대 직장인 고객이 방문했습니다.
현재 보유자산 3천만원, 중립적 투자 성향을 가지고 있습니다.
5분 설문을 통해 맞춤형 포트폴리오를 즉시 제안해드리겠습니다."
```

**화면 조작**:
- 홈 화면 → "포트폴리오 추천 시작" 버튼 클릭
- 설문 페이지 진입

---

### ⏱️ STEP 2: 실시간 설문 입력 (40초)

**발표자 멘트**:
```
"5단계 설문으로 고객 프로파일을 수집합니다.

1단계: 투자금액 - 1천만원~3천만원 선택
2단계: 위험 성향 - 중립적 선택
3단계: 투자 기간 - 중기 1~3년 선택
4단계: 선호 섹터 - 기술/IT, 헬스케어 선택
5단계: ISA 계좌 정보 - 나이 40세, 연소득 5천만원 이하 입력

기본값이 자동 선택되어 있어 빠른 진행이 가능합니다."
```

**화면 조작**:
- 빠르게 5단계 설문 완료 (실제로는 기본값 활용)
- "포트폴리오 추천 받기" 버튼 클릭

**기술 강조 포인트**:
> "설문 완료 즉시 pykrx API를 통해 한국거래소에서 실시간 시세를 가져옵니다"

---

### ⏱️ STEP 3: AI 하이브리드 추천 결과 (60초)

**발표자 멘트**:
```
"AI 하이브리드 알고리즘이 작동합니다.

[포트폴리오 요약 카드 가리키며]
- 위험도: 중립적 (5/10점)
- 예상 수익률: 연 8-12%
- 추천 종목: 6개 (대형주 4개 + 중형주 2개)

[AI 추천 섹션으로 스크롤]
이제 핵심인 AI 추천 결과를 보겠습니다.

[추천 종목 카드들 가리키며]
- 삼성전자 (88점/100) - 강력 추천 (초록색)
- NAVER (76점/100) - 추천 (파란색)
- 카카오 (62점/100) - 권장 (파란색)

점수 산출 방식을 설명드리면..."
```

**[화면 전환: AI 알고리즘 다이어그램 슬라이드]**

```
"3가지 알고리즘을 조합한 하이브리드 방식입니다:

1. Content-Based Filtering (50%)
   → 28차원 특징 벡터로 유사도 계산
   → PER, PBR, ROE 같은 펀더멘털 지표
   → RSI, 이동평균 같은 기술적 지표
   → 고객 프로파일과 가장 유사한 종목 추천

2. Collaborative Filtering (30%)
   → 12명의 검증된 투자자 패턴 학습
   → 유사한 성향 투자자들이 보유한 종목 추천
   → Jaccard Similarity로 패턴 매칭

3. Popularity-Based (20%)
   → 가장 많이 보유된 안정적 종목
   → 시장 검증된 선택지 제공

최종 점수 = (CB * 0.5) + (CF * 0.3) + (POP * 0.2)
```

**기술 강조 포인트**:
> "scikit-learn의 cosine_similarity와 euclidean_distance를 활용한 진짜 머신러닝 알고리즘입니다"

---

### ⏱️ STEP 4: 리밸런싱 제안 (40초)

**[화면: 나의 투자 관리 섹션으로 스크롤]**

**발표자 멘트**:
```
"이제 실전입니다. 고객의 현재 보유 자산을 입력합니다.

[개인 자산 관리 섹션 가리키며]
- 삼성전자: 10주 보유, 매수가 70,000원
  → 현재가 72,500원 (실시간)
  → 평가손익: +250,000원 (+3.6%)

[목표 비중 vs 현재 비중 비교]
AI 추천 포트폴리오:
- 삼성전자 30% 권장 → 현재 25% 보유
- NAVER 20% 권장 → 현재 10% 보유
- 카카오 15% 권장 → 미보유

[리밸런싱 제안]
▶ 삼성전자 2주 추가 매수 (약 145,000원)
▶ NAVER 5주 신규 매수 (약 850,000원)
▶ 카카오 10주 신규 매수 (약 450,000원)

→ 상담 중 즉시 구체적인 매매 제안 가능!"
```

**기술 강조 포인트**:
> "실시간 KRX 시세로 정확한 금액을 자동 산출합니다"

### ⏱️ STEP 5: 금융 이론 검증 (40초)

**[화면: MPT 분석 섹션으로 스크롤]**

**발표자 멘트**:
```
"단순한 추천이 아닙니다. 노벨상 수상 이론으로 검증합니다.

[MPT 효율적 투자선 차트 가리키며]
- 1,000개의 무작위 포트폴리오 시뮬레이션
- 빨간 점: 추천 포트폴리오 위치
- 위험 14.2% 대비 예상수익률 10.8%
- 효율적 투자선 위에 위치 = 최적 배분

[백테스팅 차트 가리키며]
- 최근 1년 히스토리컬 데이터 검증
- KOSPI 대비 +3.2%p 초과 수익
- 최대 낙폭(MDD) 12.5% = 안정적
```

**[화면: 뉴스 감성 분석 섹션 잠깐 보여주기]**

```
"추가로 최신 뉴스 5건을 AI로 감성 분석합니다.
긍정/부정 키워드 기반으로 -100~+100점 산출
→ 삼성전자 +45점 = 긍정적 시장 분위기"
```

**기술 강조 포인트**:
> "해리 마코위츠의 Modern Portfolio Theory (1990년 노벨 경제학상)를 실제 구현했습니다"

## 💡 데모 진행 팁

### DO ✅
1. **속도 조절**: 5분을 넘기지 말 것 (Q&A 시간 필요)
2. **실시간 강조**: API 호출 순간 강조 (로딩 인디케이터 보여주기)
3. **숫자 강조**: 88점, 3.6% 수익률, +145,000원 등 구체적 수치
4. **화면 전환 최소화**: Results 페이지 내에서 스크롤만 활용
5. **마우스 하이라이트**: 중요한 섹션은 마우스로 동그라미 그리기

### DON'T ❌
1. **용어 설명 길게**: "Cosine Similarity는..." 같은 설명 자제
2. **코드 보여주기**: 데모 중에는 코드 NO (Q&A에서만)
3. **실패 시나리오**: 로딩 실패 등 부정적 상황 언급 NO
4. **기능 나열**: "이것도 있고 저것도 있고" 식 나열 NO
5. **경쟁 비교**: 다른 서비스와 비교 NO
